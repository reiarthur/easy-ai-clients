# video.video_with_audio(api="together")

Uses Together AI's `/v1/videos` workflow with a source `video_url` and prompt to
generate/add audio for a video.

Environment variable: `TOGETHER_API_KEY`

Default model: `Wan-AI/Wan2.2-V2A-A14B`

Cost: `cost_source="unavailable"` unless Together returns stable usage/cost
metadata.

# image.edit(api="deepinfra")

DeepInfra prompt-guided image editing through an OpenAI-compatible image edit
gateway.

Environment variable: `DEEPINFRA_API_KEY`

Default model: `black-forest-labs/FLUX-1-Kontext-dev`

Returns both `cust_usd` and `cost_usd`; cost is unavailable when provider usage
is not returned.
